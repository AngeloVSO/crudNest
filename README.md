## Descrição

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

CRUD desenvolvido para praticar o NEST, com banco noSql MongoDB.

## O que funciona:

```bash
Criar Usuário
Acessar Usuário
Editar Usuário
Deletar Usuário
```

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
